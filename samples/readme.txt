○ソフト紹介
　本サンプルソースは、「PerlフレームワークCatalyst完全入門」（インプレス
　ジャパン刊）で紹介されているソースコードを電子ベースで提供するものです。
　使用するにあたって必要な設定、及び、必要なコンポーネント類等は一切含んで
　いないので注意してください。
　本サンプルはWindows環境に最適化しています。他プラットフォームをご使用の方は
　適宜パスや文字コード等、変更した上でご使用ください。

○サンプルのテスト環境
　本サンプルは、以下の環境で動作検証を行っております。
　　・Windows XP Professional Edition SP3／Active Perl 5.8.9 Build 827
　　　／Catalyst 5.80021
　　・Windows 7 Ultimate Edition／Active Perl 5.8.9 Build 827
　　　／Catalyst 5.80021
　　・Fedora 12／Perl 5.10.1／Catalyst 5.80021
　　・MySQL 5.1.45

○参考図書
　アプリケーション開発に必要な知識を習得したいという方には、
　以下の書籍をお奨めします

　1.「基礎Perl」 （インプレスジャパン）
　　　http://www.wings.msn.to/index.php/-/A-03/978-4-8443-2699-1/
　　　Catalystの基盤言語であるPerlの入門書です。
　　　基本文法からオブジェクト指向構文、メジャーなモジュールの用法、
　　　データベース連携などなど、モダンなPerlのコーディングスタイルを
　　　基礎から徹底解説。
　　　本格的なアプリケーション開発を行うための地盤固めとして、
　　　基本的な知識をきっちりと身につけることができます。


　2.「Pocket詳解Perl/CGI辞典」 （秀和システム）
　　　http://www.wings.msn.to/index.php/-/A-03/4-7980-1297-1/
　　　ポケットサイズなのに情報量満載。Perl／CGI利用者必携のリファレンス。
　　　CGI／Template-toolkit／DBI／EncodeなどCGI開発で多用されるモジュールや
　　　関数・演算子・メソッドなどを多数収録。
　　　機能引き目次と用例で、使い方がひと目でわかります。

　3.「今すぐ使えるかんたんプラス　3ステップでしっかり学ぶMySQL入門」
　　　（技術評論社）
　　　http://www.wings.msn.to/index.php/-/A-03/978-4-7741-4076-6/
　　　本書で扱ったデータベース、MySQLの入門書です。
　　　「予習」「体験」「理解」の3ステップ構成で、しっかり確実に
　　　ステップアップできます。
　　　習熟度が確認できる練習問題付きで、解答は取り外し可能。
　　　インストールファイル一式とサンプルファイルの入ったCD-ROM付きです。
